# 币安策略仪：回测
SONGGUOPENG.TradingBack 2026元旦节版本：集成“手动玩耍”模拟交易游戏到本软件（主窗口点击‘运行’，点击‘手动玩耍’）。

本游戏优点：

1.三条曲线同时绘制且实时刷新（K线蜡烛图、成交量柱状图、资金折线图）；

2.左侧实时刷新可用资金、余额资金、持仓价值、浮动盈亏、预估强平价、持仓均价；

3.K线蜡烛图加载三条虚线（均价、止盈价、止损价）；

4.随意调节K线推进速度，最高每秒1000根；

5.K线数据来自于Excel工作表，由SONGGUOPENG.BinanceKline获取交易所历史K线，最高粒度1minute；

6.保留主界面的后备隐藏能源系统；

7.基于SONGGUOPENG.Software开发，更多功能扩展至主窗口；

8.点击关闭图形绘制可以加速K线推进速度，而左侧依旧维持动态实时刷新；

9.对标交易所模拟仓位，但本游戏体验更佳、功能更丰富且不需要注册账号；

10.本游戏完全免费；
