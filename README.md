# 币安策略仪：回测
本软件可以针对加密货币的历史价格进行策略测试，参数设置的变化影响收益的走势。
# 软件优点：
1.功能复杂但使用简便。
2.回测数据集真实且极大。
3.数据集也可任意选择。
4.曲线图更直观。
5.中间过程可详细保存。
6.不同参数的资金曲线图可集中展示对比。
7.永久免费。
